Resumo

O trabalho aqui introduzido aborda a utilização de servidores web para a gestão de uma aplicação para submissão de teses.

Para este trabalho foram desenvolvidos dois tipos de ficheiros: um para o servidor e outro para o estilo das páginas da aplicação gerada.

Relativamente ao servidor, este foi desenvolvido para receber pedidos na porta 4006. Este recebe vários tipos de pedidos diferentes de dois métodos diferentes: GET e POST. Relativamente ao método GET, o servidor pode aceitar pedidos para aceder à página inicial, registar e processar o registo de uma tese, listar todas as teses e solicitar o ficheiro css de estilo da página. Relativamente ao método POST, o servidor pode aceitar pedidos para processar o registo de uma tese. Aqui, o registo é guardado num ficheiro JSON.

Quanto aos ficheiros de estilo, estes foram desenvolvidos em pug e definem o estilo da página inicial, de registo, de processamento do registo, de listagem das teses e de erro.

Resultados
