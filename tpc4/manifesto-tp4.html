Resumo

O trabalho aqui introduzido aborda a utilização de servidores web para a gestão de um website sobre um Arquivo de Música Digital.

Inicialmente foi feita a especificação de uma stylesheet para a geração do website. Este contém um índice com todas as obras existentes ordenadas pelo seu tipo. Cada obra possui uma página html com as informações da mesma. Através do índice é possível aceder às diferentes obras através de um link do tipo http://localhost:1150?id=m_id. O índice está referenciado com um link do tipo http://localhost:1150/obras. Desta forma, vamos utilizar estes links para, através do servidor, devolver as páginas quando solicitadas.

O servidor quando recebe um pedido, faz parse do url e, dependendo do seu tipo, devolve a página da obra ou do índice correspondente.

É possível aceder aos ficheiros no link https://github.com/a78275/pri.git.
