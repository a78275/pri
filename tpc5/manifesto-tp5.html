Resumo

O trabalho aqui introduzido aborda a utilização de servidores web para a gestão de um website sobre um Arquivo de Música Digital.

Para este trabalho foram desenvolvidos dois tipos de ficheiros: um para o servidor e outro para o estilo das páginas do website geradas.

Relativamente ao servidor, este foi desenvolvido para receber pedidos na porta 6600. Este recebe 3 tipos de pedidos diferentes. Um deles quando o utilizador solicita a página de índice (esta página é criada em formato XML quando o servidor é inicializado), outro quando solicita uma das obras do índice, e finalmente quando cada uma destas solicita o ficheiro css.

Quanto aos ficheiros de estilo, estes foram desenvolvidos em pug e definem o estilo da página de índice e da página de cada obra.

Resultados

  1. Servidor Node.js.
  2. Página inicial com o índice do website.
  3. Link do repositório git onde se encontra o trabalho.
